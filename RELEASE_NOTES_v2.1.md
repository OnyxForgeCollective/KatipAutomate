### 🤖 Intelligent Typing Features

#### 4. **Human-Like Typing Mode**
New toggle: **"🤖 İnsan Gibi Yaz"**

When enabled, the bot types more naturally:
- 🎲 **Random speed variations** (±20% from base speed)
  - Base: 120ms → Actual: 96-144ms
- ⏸️ **Random pauses** (5% chance)
  - Duration: 200-800ms
  - Simulates thinking/hesitation
- 🧠 More human behavior patterns

**Why use it?**
- More natural typing patterns
- Harder to detect as automation
- Mimics real human behavior

---

#### 5. **Advanced Mistake System**
New dropdown: **"❌ Hata Modu"**

Three modes available:
1. **Kapalı** - No mistakes (default)
2. **Basit Mod** - Simple mistakes
3. **Gelişmiş Mod** ⭐ - Advanced intelligent mistakes

**Gelişmiş Mod Features:**
- 🎯 Makes mistakes on every Nth word (configurable)
- ⌨️ Realistic typos based on keyboard proximity
  - Example: 'a' → 's' (adjacent key)
- 🎲 70% chance to notice and correct
- ⏱️ Natural correction timing:
  - Pause (200-500ms) → Backspace → Correct letter
- 🔄 30% chance to leave the mistake

**Configuration:**
- Slider: "Her kaç kelimede bir" (2-10 words)
- Visual feedback in UI
- Info box explaining the mode

**Example Sequence:**
```
Typing "hello world example"
1. "hello" - correct
2. "world" - correct
3. "examoke" (mistake on 3rd word)
   - Pause 350ms
   - Backspace: "examok"
   - Type "p": "exampke"
   - Backspace: "examk"
   - Type "l": "examle"
   - Backspace: "exam"
   - Type "ple": "example"
```
